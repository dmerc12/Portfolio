## Getting Started

### Initial Setup
1. **Setup a PostgreSQL database**

2. **Clone the repository:**
    ```sh
    git clone https://github.com/dmerc12/Portfolio.git
    cd Portfolio
    ```

3. **Make the scripts executable:**
    ```sh
    chmod +x start.sh stop.sh setup.sh
    ```

4. **Setup and start the application:**
    ```sh
    ./setup.sh
    ```

### Manage Application
1. **Stop the application:**
    ```
    ./stop.sh
    ```

2. **Start the application after initial setup:**
    ```
    ./start.sh
    ```
